# API Quick Reference Guide

**Base URL:** `http://localhost:8080/api/v1/admin`

---

## üöÄ Quick Start

### Create a Salesman (Example)
```bash
curl -X POST http://localhost:8080/api/v1/admin/salesmen \
  -H "Content-Type: application/json" \
  -d '{
    "firstName": "John",
    "lastName": "Doe",
    "address": "New York",
    "contactNumber": "+919876543210"
  }'
```

### Create a Vendor
```bash
curl -X POST http://localhost:8080/api/v1/admin/vendors \
  -H "Content-Type: application/json" \
  -d '{
    "vendorName": "ABC Chemicals",
    "firstName": "Rajesh",
    "lastName": "Kumar",
    "address": "Dhanbad",
    "contactNumber": "+919876543210"
  }'
```

### Create a Customer
```bash
curl -X POST http://localhost:8080/api/v1/admin/customers \
  -H "Content-Type: application/json" \
  -d '{
    "shopName": "Sharma General Store",
    "ownerFirstName": "Priya",
    "ownerLastName": "Sharma",
    "ownerAddress": "Hazaribagh",
    "customerType": "Shopkeeper",
    "routeId": 1,
    "villageId": 1
  }'
```

## üîë Key Fields by Entity

### Salesman
- `firstName` (required)
- `lastName` (required)
- `address` (required)
- `contactNumber` (optional)

### Vendor
- `vendorName` (required)
- `firstName` (required)
- `lastName` (required)
- `address` (required)
- `contactNumber` (optional)

### Customer
- `shopName` (required)
- `ownerFirstName` (required)
- `ownerLastName` (required)
- `ownerAddress` (required)
- `customerType` (required): Dealer, Subdealer, Shopkeeper, Household
- `routeId` (optional)
- `villageId` (optional)

### Product
- `productName` (required)
- `variant` (optional)
- `size` (optional)
- `targetPrice` (optional)
- `baseCommission` (optional)
- `otherOverheadCost` (optional)

### Chemical
- `chemicalName` (required)
- `category` (required): Raw Material, Packaging, Labeling, Other
- `unit` (optional)
- `purchaseRate` (optional)
- `transportCostPerUnit` (optional)
- `vendorId` (optional)

```json
  "createdAt": "2026-02-19T12:30:00"
}
```

## üîó Important Notes

- **No Manual IDs:** All IDs are auto-generated by the backend
- **No Authentication:** All endpoints are publicly accessible
- **Timestamps:** All created records have `createdAt` timestamp
- **CORS:** Frontend on `localhost:3000` is whitelisted
- **Content-Type:** Always send `application/json`
